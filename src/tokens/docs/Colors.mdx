# Colors

Colors are defined in [`src/tokens/colors.ts`](?path=/story/foundations-design-tokens-colors--docs)
and generated as `--color-*` CSS custom properties by `npm run generate:tokens`.

## Palettes

Raw scales. Only use when a semantic role doesn't fit; prefer the
semantic roles below for anything component-facing.

The role-based tokens components consume — these map onto the raw tokens
above, so changing a raw token ripples through.
