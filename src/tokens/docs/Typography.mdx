# Typography

Typography tokens live in [`src/tokens/typography.ts`](../colors.ts) and are generated as
`--font-*`, `--line-height-*`, and `--letter-spacing-*` CSS variables. The
composed **text styles** are what the [`Text`](?path=/docs/foundations-text--docs)
component uses via `.text-*` classes.

## Font families

- **sans** — `"Inter", system-ui, sans-serif` — body & UI
- **serif** — `"Libre Baskerville", Georgia, serif` — headings
- **mono** — `"JetBrains Mono", "Courier New", monospace` — code & numerics
